# Support Policy

## Security Support

MilliWays security support is provided for the current stable release line.

Security support until: 2029-12-31

Supported security surfaces include the terminal launcher, daemon RPCs, runner handoff, session storage, installer scripts, package release artifacts, and Secure MilliWays scanner integration on macOS and Linux.

Report vulnerabilities through the process in [SECURITY.md](SECURITY.md).
